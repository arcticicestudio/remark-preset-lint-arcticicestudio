Testing

Please make sure to complete the requirement and installation steps first in order to test the project!

Continuous integration builds are running at Circle CI.

Linting

The package "lints itself" using and extending the defined rules:

npm run lint:md

JavaScript

ESLint is used in order to check all JavaScript sources using the Arctic Ice Studio ESLint Base Config:

npm run lint:js
